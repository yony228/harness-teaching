## 四、需要调整的事项

1. **Organizer 增加文件校验**：在写入后验证 JSON 格式合法性，避免写入损坏数据
2. **多源去重逻辑**：当来自不同采集时间（多文件）时，需基于 `source_url` 去重
3. **Analyzer 输出结构化**：当前为文本格式输出，后续可考虑返回结构化 JSON 方便 Organizer 消费
4. **Agent 之间增加状态传递**：当前为手动复制粘贴，理想状态是通过 LangGraph 状态机传递分析结果
